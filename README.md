# Shell Setup

## Homebrew packages

```
brew install navi antidote
```

## Update antidote bundle

```
antidote bundle <~/.zsh_plugins.txt >~/.zsh_plugins.zsh
antidote bundle <~/.zsh_plugins.post_compinit.txt >~/.zsh_plugins.post_compinit.zsh
```
